# Une application MERN 

### Objectif

Codex est une application de blog centrée sur les articles de technologie. Elle utilise la pile MERN (MongoDB, Express, React, Node.js) pour fournir un backend puissant et un frontend interactif. L'application permet aux utilisateurs de publier, lire et commenter des articles sur des sujets liés à la technologie.

### Prérequis

Avant de commencer, vous devez installer les outils suivants sur votre machine :

-Node.js et npm  : Télécharger Node.js

-MongoDB : Utilisez une instance locale de MongoDB ou un service comme MongoDB Atlas pour la version cloud.

-Docker (facultatif, mais recommandé pour la production) : Télécharger Docker


### Démarrer l'application avec Docker

`docker-compose build`

###Démarrer les services

`docker-compose up`
